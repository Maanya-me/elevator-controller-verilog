# Elevator Controller (Verilog)

This project implements a multi-floor elevator controller using a Finite State Machine (FSM) in Verilog.

## Features
- Handles floor requests
- Controls elevator movement
- Door open/close logic
- Simulation testbench included

## Files
- elevator_controller.v : main Verilog module
- elevator_controller_tb.v : testbench for simulation

## Concepts Used
- Finite State Machine (FSM)
- Sequential Logic
- Digital System Design
